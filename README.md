## ⚙️ How to Use velboard

Once velboard is running, you will see nine main panels. Each panel shows live data about OpenClaw agents. The data updates automatically, so you can watch things live.

Here is what each panel focuses on:  

- Agent Status: Shows if agents are online or offline.  
- Activity Log: Displays recent actions by the agents.  
- Claude Integration: Tracks Claude AI agents’ current state.  
- Telegram Panel: Shows messages or commands sent via Telegram.  
- Performance Metrics: CPU and memory use of agents.  
- Errors and Warnings: Lists any problems found.  
- Connectivity: Network status of connected agents.  
- Command History: Commands sent to OpenClaw agents.  
- Overall Health: A quick view of system status.

You can move or resize panels to see your preferred info clearly.

## 🔧 Troubleshooting Tips

If velboard does not start or shows errors, try these:

- Restart your computer and try again.  
- Check your internet connection. velboard needs to contact agents.  
- Make sure your antivirus software is not blocking velboard.  
- Download velboard again from the release page to get the latest version.  
- Run velboard as administrator by right-clicking the icon and selecting "Run as administrator."
